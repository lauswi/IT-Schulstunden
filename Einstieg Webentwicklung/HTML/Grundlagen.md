## Aufbau einer Webseite

Es gibt im HTML Code immer einen Start-Tag und einen End-Tag. 
Bsp.: 

```html run

<html>
  <head>
        <title> Meine erste Webseite </title>
  </head>
  <body>
  <!--Das ist ein Kommentar-->
  <p> Hallo Webwelt! </p>
  
  </body>
</html>
```
